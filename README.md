# Snap-View v.1.2
Snap View is a lightweight program that allows you to view the contents of various types of ASCII files

# Main features
* Quick file opening.
* Syntax highlight for supported file types.
* Text search capability within the open file.
* Option to keep the application always on top.
* Displays CSV files in text and grid formats.
* For some file types, can show a list of funciones/procedures.
* You can also associate Snap View as the default viewer (F3) for TotalCommander, FreeCommander, or another file manager.
* Works on Windows 7, 8, and 10.
* Snap View is freeware.
